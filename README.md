# Английская и русская раскладка для Linux

## Установка

```
git clone https://github.com/vovanmozg/universal-layout-linux.git
cd universal-layout-linux
sudo bash bin/install.sh
``` 
Затем добавить раскладку в System Settings / Keyboard
